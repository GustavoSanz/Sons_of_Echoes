# Sons Of Echoes 

## Descrição
Sons Of Echoes é um jogo de plataforma 2D focado em exploração, combate mágico e gestão de recursos (Vida e Mana). O jogador controla o mago Orin através de ambientes misteriosos, enfrentando inimigos e interagindo com o ambiente.

## Tecnologias Utilizadas
* **Motor de Jogo:** Phaser 3 (HTML5 / JavaScript)
* **Linguagens:** HTML, CSS, JavaScript

## Mecânicas Principais
* **Combate Dinâmico:** Sistema de projéteis (orbes mágicos) que consome mana.
* **Inteligência Artificial:** Inimigos reactivos baseados na distância e linha de visão do jogador.
* **Gestão de Recursos:** Estátuas interativas no mapa permitem recarregar a mana com tempos de cooldown precisos.
* **Sistema de Inventário:** UI dedicada para visualização e uso de itens de cura.
* **Transições e Diálogos:** Sistema de diálogos estilo "Typewriter" e transições de ecrã dinâmicas.

## Controlos
* **Setas Direcionais:** Mover a personagem (Esquerda / Direita) e Saltar (Cima).
* **Seta Baixo:** Agachar / Interagir com o ambiente (ex: Estátuas de Mana).
* **Shift:** Correr.
* **J:** Atirar Magia (Consome 3 pontos de Mana).
* **I:** Abrir / Fechar Inventário.
* **U:** Usar Item de Cura através do inventário.

1. Iniciar um servidor local na pasta raiz do projeto (ex: extensão *Live Server* do VSCode, XAMPP ou Python `http.server`).
2. Abrir o ficheiro `index.html` no navegador.
